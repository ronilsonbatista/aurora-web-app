# Aurora & Co.

Site institucional da Aurora & Co.

Uma marca premium focada em gestão estratégica, posicionamento digital, branding, operação e crescimento de criadoras.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion

## Design

- Luxo minimalista
- Editorial premium
- Paleta vinho, preto e off-white
- Experiência responsiva

## Estrutura

- Home
- Sobre
- Gestão
- Estrutura
- Modelos
- Contato

## Desenvolvimento

```bash
npm install
npm run dev
```
